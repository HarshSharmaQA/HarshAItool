# **App Name**: Stratic CMS

## Core Features:

- Admin Authentication: Secure admin panel with login and role-based access control.
- Dynamic Page Generation: Generate static pages from Firestore data using slugs.
- Blog Post Management: Create, update, and delete blog posts via the admin panel and API routes.
- Menu Management: Manage site menus (header, footer) via a UI in the admin panel.
- Global Settings: Manage site-wide settings (title, logo) via a UI.
- API Routes for CRUD: Secure server-side API routes for all create, read, update, and delete operations on Firestore data, with admin authentication check on write operations.
- AI-Powered Content Suggestions: The create/edit content forms will feature an AI-powered tool to suggest content based on the current copy in the editor.

## Style Guidelines:

- Primary color: Deep purple (#6750A4), evoking sophistication and trustworthiness.
- Background color: Light lavender (#E6E0EB), providing a gentle contrast and modern feel.
- Accent color: Soft blue (#A0AEC0), for highlighting interactive elements and calls to action.
- Headline font: 'Playfair', serif, with an elegant and high-end feel, suitable for headlines.
- Body font: 'PT Sans', sans-serif, which provides a modern look and clear legibility for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a consistent set of minimalist icons for navigation and content actions.
- Maintain a clean and responsive layout, with clear visual hierarchy to facilitate content consumption and ease of navigation.
- Use subtle animations for transitions and feedback on user interactions to improve engagement and visual appeal.
